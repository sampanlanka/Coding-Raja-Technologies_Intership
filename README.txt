Object Detection and Tracking in Videos :

This project focuses on developing a robust system capable of detecting and tracking objects of interest in video streams using advanced deep learning and computer vision techniques. The main objectives of this project are to detect objects in real-time video streams and track them across consecutive frames

Chatbot using Natural Language Processing (NLP) :

This project revolves around the development of a conversational chatbot capable of understanding and responding to user input using natural language processing (NLP) techniques. The chatbot's primary goal is to engage users in meaningful conversations and provide relevant responses based on their queries.
